# Breast_Cancer_Detection
A Machine Learning web application that predicts breast cancer malignancy with 96.5% accuracy using Random Forest and Flask. Trained on the Wisconsin Breast Cancer Dataset.


The Breast Cancer Prediction System is a Machine Learning web application designed to assist medical professionals in the early detection of breast cancer. Built using Python and Flask, the system utilizes a Random Forest Classifier trained on the Wisconsin Breast Cancer Dataset to predict malignancy with 96.5% accuracy. It features a user-friendly interface that processes cell nucleus data to provide instant diagnostic insights, helping to reduce the time and cost associated with traditional testing methods
